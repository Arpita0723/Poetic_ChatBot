# 🌸 Poetic Personality Chatbot

This project is a Streamlit-based chatbot that generates poems based on user input using Google's Gemini AI model. The chatbot takes a user's thoughts or message as input and responds with a creative poem in a natural and interactive way.

---

## Features

* Generate poems using Google's Gemini AI
* Simple and interactive Streamlit interface
* Instant poem generation based on user input
* Easy to use and lightweight application

---

## Tech Stack

* Python
* Streamlit
* Google Gemini API
* Google Generative AI SDK

## How to Run

Clone the repository

```bash
git clone https://github.com/Arpita0723/Poetic_ChatBot.git
```

Go to the project folder

```bash
cd Poetic_ChatBot
```

Install the required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

---

## How It Works

1. Enter a message or idea in the input box.
2. The application sends the input to the Gemini API.
3. Gemini generates a poem based on the given prompt.
4. The generated poem is displayed instantly in the Streamlit interface.
